Values and Units 2

Your file for this exercise should be saved in the public folder and should be accessed using this link: "/css-exercises/values-and-units-2.html". The content of the HTML file is given below. With the given HTML content, design the web page using an internal style sheet in such a way it will look exactly like the screenshot below.
Note: You cannot modify the content, only the style sheet!
Aside from the design visible in the screenshot, here are some additional details for the design of the web page:
  • The background color of the body should be #888888
  • The font color of the subtitle should be gray
  • The line height of the title and subtitle should be 10px
  • The poem should have a width of 30% and left and right padding of 10px,
  • The font size of the first paragraph should be 1.5 times the size of the current font
  • The line height of the third and fourth paragraph should be 20/72 in
  • Use ex unit to convert the strong elements as a subscript
