Selectors 1

CSS is a style sheet language developed to control the presentation of markup language documents. HTML is to content while CSS is to design. Using CSS, you can:
  • change the style and layout of your page without changing its content
  • use one or a small number of CSS files to control how your whole website will look like
  • define separate styles for different kinds of media without changing the structure of your website
A CSS rule set consists of a selector and a declaration block. The selector points to the HTML element you want to style. The declaration block contains one or more declarations separated by semicolons. Each declaration includes a property name and a value, separated by a colon.

Example

h1 {color: blue;} 
h1 {color: blue;}

Headlines or Headings

Headline or heading tags are usually used to denote a title or subtitle. There are six levels of headlines, ranging from <h1>…</h1> to <h6>…</h6> with the <h1> tag being the largest and <h6> tag being the smallest. Each time you place a heading tag, your web browser automatically places a line break after your ending tag.

Example

Level 1 Headline

Level 2 Headline

Level 3 Headline

Level 4 Headline

Level 5 Headline
Level 6 Headline
<h1>Level 1 Headline</h1>
<h2>Level 2 Headline</h2>
<h3>Level 3 Headline</h3>
<h4>Level 4 Headline</h4>
<h5>Level 5 Headline</h5>
<h6>Level 6 Headline</h6>
