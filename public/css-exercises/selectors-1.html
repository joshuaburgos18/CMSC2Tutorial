Selectors 1

CSS is a style sheet language developed to control the presentation of markup language documents. HTML is to content while CSS is to design. Using CSS, you can:
  • change the style and layout of your page without changing its content
  • use one or a small number of CSS files to control how your whole website will look like
  • define separate styles for different kinds of media without changing the structure of your website
A CSS rule set consists of a selector and a declaration block. The selector points to the HTML element you want to style. The declaration block contains one or more declarations separated by semicolons. Each declaration includes a property name and a value, separated by a colon. There are various properties that can be used to set the display of HTML elements, and they are easy to remember since property names are self-descriptive.

Example

Hello World!

How are you?
<!DOCTYPE html>
<head>
	<title>CMSC 2 Example</title>
</head>
<body>
	<h1>Hello World!</h1>
	<h6>How are you?</h6>
</body>
h1 {color: blue; text-align: center}
The example above sets the display of all <h1> tags in the web page. The color and alignment properties are modified to blue and center, repectively.

Ways to Insert CSS

There are three ways of inserting a style sheet:
  • External Style Sheet
  • Internal Style Sheet
  • Inline Style

External Style Sheet
An external style sheet can be written in any text editor. The file should not contain any html tags, only the CSS rules. The style sheet file must be saved with a .css extension. With an external style sheet, you can design an entire website by changing just one file.

To design an entire website using an external style sheet, each web page must include a reference to the external style sheet using the <link> element. The <link> element goes inside the <head> section. For example, mystyle.css is the name of the style sheet you want to use in your web page.

Example

<head>
	<link rel="stylesheet" type="text/css" href="css/mystyle.css">
</head>
The example above used a relative path to state the location of the style sheet which is inside a folder named css. The folder is located in the same location of the HTML web page.

Internal Style Sheet
Internal styles are defined within the <style> element, inside the <head> section of an HTML page. With an internal style sheet, you can design a single page.

Example

<head>
	<title>CMSC 2 Example</title>
	<style>
		body {
			background-color: yellow;
		}

		h1 {
			color: #FF0000;
			margin-left: 40px;
		} 
	</style>
</head>

Inline Style
Inline styles are added using the style attribute to any HTML tag. An inline style may be used to apply a unique style for a single element. The style attribute can contain any CSS property.

Example

<img src="/images/baby.jpg" style="width:150px;height:100px;"/>

Multiple Style Sheets

Multiple Style Sheets can be included in a single web page. But what style will be used when there is more than one style specified for an HTML element? For example, a CSS rule for the <h1> tag is defined in an internal style sheet, however another rule for the <h1> tag is defined using inline style:

Example

<head>
	<style>
		h1 {
			font-size: 50px;
			color: yellow;
		}
	</style>
</head>
<body>
	<h1 style="text-align:center;color:blue;">Hello World!</h1>
</body>
Notice that the color property is specified in both the internal style sheet and inline style.

If more than one style is specified in a web page, all the overlapping styles will override the other depending on the order and priority of style sheet:
  1. Inline style (inside an HTML element)
  2. External and internal style sheets (in the head section)
  3. Browser default
 Inline style has the highest priority, which means that from our example above, the color of the <h1> element will be blue.

Another scenario is when the link to the external style sheet is placed below the internal style sheet in the<head> section. From the order that they are placed in the HTML file, the external style sheet will override the internal style sheet.

Example

<head>
	<style>
		h1 {
			font-size: 50px;
			color: yellow;
		}
	</style>
	<link rel="stylesheet" type="text/css" href="mystyle.css">
</head>
In the example above, if the color of the <h1> element is both defined in mystyle.css and in the internal style sheet, the defined color in the external style sheet will be used.

CSS Comments

Comments are used to explain your code, and may help you when you edit the source code at a later date. Comments are ignored by browsers. A CSS comment starts with /* and ends with */.

Pseudo-class Selectors

A CSS pseudo-class is used to style a specified special state of the element. Pseudo-class selectors are CSS selectors with a colon (:) preceding them.

Anchor Pseudo-classes
Links can be displayed depending on their current state.

Example

This is a link. 
/* if link is not yet visited */
a:link {
	color: #FF0000;
	text-decoration: none;
}

/* if link is already visited*/
a:visited {
	color: #00FF00;
}

/* if link is hovered (mouse over) */
a:hover {
	color: #FF00FF;
	text-decoration: underline;
}

/* if link is clicked */
a:active {
	color: #0000FF;
}

Note that a:hover MUST come after a:link and a:visited in the CSS definition in order to be effective. a:active MUST come after a:hover in the CSS definition in order to be effective. Pseudo-class names are not case-sensitive.

There are other various pseudo-classes that can be used to style a web page. Here is a list of the commonly used CSS pseudo-classes:

:active a:active Selects the active link
:checked input:checked Selects every checked <input> element
:disabled input:disabled Selects every disabled <input> element
:enabled input:enabled Selects every enabled <input> element
:first-child p:first-child Selects every <p> elements that is the first child of its parent
:first-of-type p:first-of-type Selects every <p> element that is the first <p> element of its parent
:focus input:focus Selects the <input> element that has focus
:hover a:hover Selects links on mouse over
:last-child p:last-child Selects every <p> elements that is the last child of its parent
:last-of-type p:last-of-type Selects every <p> element that is the last <p> element of its parent
:link a:link Selects all unvisited links
:optional input:optional Selects <input> elements with no "required" attribute
:required input:required Selects <input> elements with a "required" attribute specified
:visited a:visited Selects all visited links

Pseudo-elements

A CSS pseudo-element is used to style specified parts of an element. Pseudo-elements are CSS selectors with a double colon (::) preceding them. This was an attempt from W3C to distinguish between pseudo-classes and pseudo-elements.

The ::first-line Pseudo-element
The ::first-line pseudo-element is used to add a special style to the first line of a text. The ::first-line pseudo-element can only be applied to block elements.

Example

This is a paragraph. A very very long paragraph to give you an example of what pseudo-elements can do. I hope you enjoy the tutorials. Kthxbye!

}

There are other various pseudo-elements that can be used to style a web page. Here is a list of all the CSS pseudo-elements:
Selector Example Example description
::after p::after Insert content after every <p> element
::before p::before Insert content before every <p> element
::first-letter p::first-letter Selects the first letter of every <p> element
::first-line p::first-line Selects the first line of every <p> element
::selection p::selection Selects the portion of an element that is selected by a user
