Measurements and positions in CSS properties, such as width, margin, padding, font-size, border-width, etc., are expressed in length units. Length is a number followed by a length unit, such as 10px, 2em, etc. For some CSS properties, negative lengths are allowed. There are two types of length units: relative and absolute.

Example

This is an example of a paragraph.

p {
	margin: 0px auto;	/* this declaration makes the element center-aligned */
	border: 1px solid black;
	width: 80%;
	font-size: 3em;
	text-indent: 20px;
}


Relative Length Units

Relative length units specify a length relative to another length property. Relative length units scales better between different rendering mediums.

Unit Description
em Relative to the font-size of the element (2em means 2 times the size of the current font)
ex Relative to the x-height of the current font (rarely used)
ch Relative to width of the "0" (zero)
rem Relative to font-size of the root element
vw Relative to 1% of the width of the viewport
vh Relative to 1% of the height of the viewport
vmin Relative to 1% of viewport's smaller dimension
vmax Relative to 1% of viewport's larger dimension
%

Absolute Length Units

The absolute length units are fixed and a length expressed in any of these will appear as exactly that size. Absolute length units are not recommended for use on screen, because screen sizes vary so much. However, they can be used if the output medium is known.

Unit Description
cm centimeters
mm millimeters
in inches (1in = 96px = 2.54cm)
px pixels (1px = 1/96th of 1in)
pt points (1pt = 1/72 of 1in)
pc picas (1pc = 12 pt)
