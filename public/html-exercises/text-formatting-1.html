Text Formatting 1

HTML defines special elements for specifying how a text should appear. In this page, you’ll learn some formatting elements that will help you make your web pages a lot more exciting. Note that all formatting tags should be placed inside the <body> section.

Headlines or Headings

Headline or heading tags are usually used to denote a title or subtitle.

There are six levels of headlines, ranging from <h1>…</h1> to <h6>…</h6> with the <h1> tag being the largest and <h6> tag being the smallest. Each time you place a heading tag, your web browser automatically places a line break (new line) after your ending tag.

Example

Level 1 Headline

Level 2 Headline

Level 3 Headline

Level 4 Headline

Level 5 Headline
Level 6 Headline
<h1>Level 1 Headline</h1>
<h2>Level 2 Headline</h2>
<h3>Level 3 Headline</h3>
<h4>Level 4 Headline</h4>
<h5>Level 5 Headline</h5>
<h6>Level 6 Headline</h6>

Paragraphs and Line Breaks

HTML content can be divided into paragraphs using the <p> tag. It adds space between your paragraphs.

Example

This is a paragraph.

This is another paragraph.

This is the last paragraph.

<p>This is a paragraph.</p>
<p>This is another paragraph.</p>
<p>This is the last paragraph.</p>

If you want a line break (new line) without starting a new paragraph, then the <br> tag is used. The <br> tag is empty, which means it does not need a closing tag. However, for XHTML validity, you can add a slash before the greater-than sign.

Example

We
need
a
break. 
We<br/>need<br/>a<br/>break.

Emphasis, Strong and Horizontal Rules

The <em> tag defines emphasized text while the <strong> tag defines strong text.

Example

This text is strong. This text is emphasized. 
This text is <strong>strong</strong>.
This text is <em>emphasized</em>.

The <hr> tag creates a horizontal line in an HTML page. Similar to the <br> tag, the <hr> tag does not need a closing tag.

Example

This is you.

This is me.

<p>This is you.</p>
<hr/>
<p>This is me.</p>

Quotes

The <q> tag defines a short quotation.When a <q> tag is used, the browser usually inserts quotation marks around the element.

Example

Victory's goal is to: Honor God and make disciples.

<p>Victory's goal is to: <q>Honor God and make disciples.</q></p>

The <blockquote> tag defines a long quotation. When it is used, the browser usually indents the element.

Example

The following description of XHTML is taken from the W3C Web site:

XHTML 1.0 is the W3C's first Recommendation for XHTML, following on from earlier work on HTML 4.01, HTML 4.0, HTML 3.2 and HTML 2.0 
<p>The following description of XHTML is taken from the W3C Web site:</p>
<blockquote>
	XHTML 1.0 is the W3C's first Recommendation for XHTML, following on from earlier work on HTML 4.01, HTML 4.0, HTML 3.2 and HTML 2.0
</blockquote>

Preformatted Text

The <pre> tag defines preformatted text. Text in a <pre> element is displayed in a fixed-width font (usually Courier), and it preserves both spaces and line breaks.

Example

H     E     L     L     O


W     O     R     L     D
<pre>
H     E     L     L     O


W     O     R     L     D
</pre>

Abbreviations, Addresses and Citations

The <abbr> tag defines an abbreviation or an acronym. The title attribute is used to define the meaning of the abbreviation or acronym. In the example below, if you hover on to the text UP, the meaning of the abbreviation is shown.

Example

UP was founded in 1908.

<p><abbr title="University of the Philippines">UP</abbr> was founded in 1908.</p>

The <address> tag defines contact information (author/owner) of a document or article. It is usually displayed in italics.

Example

Institute of Computer Science, Physical Sciences building, UPLB
<address>Institute of Computer Science, Physical Sciences building, UPLB</address>

The <cite> tag defines the title of a work. It is usually displayed in italics.

Example

Trees by Joyce Kilmer

<p><cite>Trees</cite> by Joyce Kilmer</p>
