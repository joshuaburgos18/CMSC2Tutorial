Conditional statements are a set of statements used to perform different actions based on different conditions. JavaScript supports two conditional statements: if-else and switch. But before we discuss them, let's first discuss some important topics such as Boolean Values, Comparison and Logical Operators.

Boolean Values

A JavaScript Boolean represents one of two values: true or false. You can use the Boolean() function to find out if an expression (or a variable) is true.
Try This Example!

Boolean Example

<script>
	function compareValues(){
		var x = Boolean(111 > 32);
		//you can also omit the keyword Boolean
		var y = (12 < 12);
		var z = (12 >= 43);
		document.getElementById('sample1').innerHTML = x + "<br/>" + y + "<br/>" + z;
	}
</script>

Comparison Operators

Comparison operators are used in logical statements to determine equality or difference between variables or values.

These are the comparison operators that you can use in JavaScript. For the examples given, let's assume that x=5.
Operator Description Example
== equal to (x == 8) //returns false
=== equal value and equal type (x === "5") //returns false
!= not equal (x != 8) //returns true
!== not equal value or not equal type (x !== "5") //returns true
> greater than (x > 8) //returns false
< less than (x < 8) //returns true
>= greater than or equal to (x >= 8) //returns false
<= less than or equal to (x <= 8) //returns true


Some things that you need to know:
  • All numberic values are evaluated as true.
  • All string values are evaluated as true.
  • Any expression (except zero) is evaluated as true.
  • Zero (0 or -0), empty string ("") and NaN are evaluated as false.

		var g;
		var h = Boolean(g);
		document.getElementById('sample2').innerHTML = w + "<br/>" + x + "<br/>" + y + "<br/>" + z;
	}
</script>

Logical Operators

Logical operators are used to determine the logic between variables or values.

These are the logical operators that you can use in JavaScript. For the examples given, let's assume that x=12 and y=22.
Operator Description Example
&& and (x > 10 && y == 10) //returns false
|| or (x != 12 || y <= 22) //returns true
! not !(x == 12) //returns false

If-Else Statement

The if statement us used to specify a block of JavaScript code to be executed if a condition is true.
If Syntax
if (condition) {
    block of code to be executed if the condition is true
}

Try This Example!

If Example

<script>
	function evalExp1(x,y){
		if(x > y){
			document.getElementById('sample3').innerHTML = x + " is greater than " + x;
		}
	}
</script>

The else statement is used to specify a block of code to be executed if the condition is false.
If-Else Syntax
if (condition) {
    block of code to be executed if the condition is true
} else {
	block of code to be executed if the condition is false
}

Try This Example!

If-Else Example

<script>
	function evalExp2(x,y){
		if(x > y){
			document.getElementById('sample4').innerHTML = x + " is greater than " + x;
		}else{
			document.getElementById('sample4').innerHTML = x + " is less than " + x;
		}
	}
</script>
